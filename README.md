### Namespaces

```
<?php
/*
 * Namespaces can be referred to as grouping which allows you to define classes and other entities under a group so that it does not conflict with  same name class in other group 
 */
namespace my\name; 

class MyClass {}
function myfunction() {}
const MYCONST = 1;
?>
```
